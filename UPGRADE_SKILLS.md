# Upgrade Skills

**For existing users:** Update your skills to the latest versions.

If you're new, see [QUICKSTART.md](QUICKSTART.md) for initial setup.

---

## How to Upgrade

1. **Fetch each SKILL.md** from the URLs below (or download the .zip directly from GitHub)
2. **Replace existing skills** in Settings > Capabilities > Skills
3. No need to re-run project setup unless your goals have changed

---

## Manual Alternative

If `skill-creator` isn't working:

1. Click each URL above to view the raw SKILL.md
2. Copy the content
3. Create a new skill manually in Claude.ai
4. Paste the content

Or use [QUICKSTART-MANUAL.md](QUICKSTART-MANUAL.md) to download .zip files directly.
